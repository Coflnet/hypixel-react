# Premium Features

SkyCofl offers several premium tiers to enhance your SkyBlock experience and support the platform's development.

## Premium Tiers

### Free
- Basic auction house browsing
- Delayed flipper (40+ seconds)
- Limited recent auctions (12)
- Basic notifications (3)
- Price history access

### Starter Premium
- Flipper delay: 10-20 seconds
- Recent auctions: 60
- Notifications: 10
- Full filter access
- Top craft/kat flips visible
- White chat color in mod

### Premium
- Flipper delay: ~1 second
- Recent auctions: 120
- Notifications: 100
- Low supply items access
- Extended flip tracking
- Priority feature requests
- Discord role
- Green chat color in mod

### Premium+
- Flipper delay: \<1 second
- All premium features
- Auction archive access
- Flip replay system
- **Bazaar data export** — data since October 2019, 20-second resolution for last 2 weeks, 5-minute resolution for older data (5 cost units per 5 minutes; full order book = 3 units, standard = 1 unit)
- **Auction archive export** — CSV export via queued job system (max 4 queued), data since October 2019
- API access: default limits still apply (`30 req/10s` and `100 req/min`); valid Premium+ token auto-unblocks IP bans
- Gold chat color in mod
- Maximum notification limits (1000)

## Frequently Asked Questions

### Can I try premium first?
Yes! Use Linkvertise for free Starter Premium or verify your Minecraft account for test premium.

### What happens if I don't renew?
You keep access until expiration, then revert to free tier. No data is lost.

### Can I change tiers?
Yes, you can upgrade anytime. Higher tiers pause lower ones while active.

### Are there discounts?
There are discounts from time to time on our subscriptions.

### Can I get refunds?
Due to the digital nature, the fact that there is free test time and immediate access, refunds are generally not available. Contact support for exceptional cases.
