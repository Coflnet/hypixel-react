# Coflnet Minecraft Mod

The Coflnet mod brings auction house data and flip notifications directly into your Minecraft client.

## Installation

### Requirements
- Minecraft 1.8.9
- Minecraft Forge for 1.8.9
- Java 8 or higher

### Step-by-Step Installation

1. **Install Minecraft Forge 1.8.9**
   - Download from [files.minecraftforge.net](https://files.minecraftforge.net/net/minecraftforge/forge/index_1.8.9.html)
   - Run the installer
   - Select "Install client"

2. **Download the Coflnet Mod**
   - Visit our [mod page](/mod)
   - Download the latest version
   - Or get it from [Modrinth](https://modrinth.com/mod/skycofl/versions)

3. **Install the Mod**
   - Locate your `.minecraft/mods` folder
   - Copy the downloaded `.jar` file into the mods folder
   - Launch Minecraft with the Forge profile

4. **First Launch**
   - Join Hypixel SkyBlock
   - Type `/cofl help` to see available commands
   - The mod will automatically connect to our servers

## Commands

### Basic Commands
- `/cofl` - Open main menu
- `/cofl help` - List all commands
- `/cofl start` - Start the mod
- `/cofl stop` - Stop the mod

### Flip Commands
- `/cofl flip` - Toggle flip notifications
- `/cofl set <setting> <value>` - Configure settings
- `/cofl filter` - Open filter menu
- `/cofl profit` - View profit statistics

### Utility Commands
- `/cofl price <item>` - Get item price info
- `/cofl ah <player>` - Open player's auctions
- `/cofl cheapest <item>` - Find cheapest listings
- `/cofl sold <item>` - See recent sales

### Advanced Commands
- `/cofl forge` - Forge recipe calculator
- `/cofl museum` - Museum item prices
- `/cofl leaderbaord` - Profit leaderboards (Premium+)
- `/cofl config` - Advanced configuration

## Troubleshooting

### Common Issues

#### Mod Not Loading
- Ensure you have Forge 1.8.9 installed
- Check that the mod file is in the correct folder
- Verify Java version compatibility
- Look for error messages in the launcher

#### Connection Problems
- Check your internet connection
- Verify Hypixel allows mod connections
- Try restarting Minecraft
- Use `/cofl start` to reconnect

#### No Flips Appearing
- Ensure you're logged in on the website
- Check your filter settings aren't too restrictive
- Verify your premium status
- Try different flip finders

#### Price Data Missing
- Wait a few minutes after joining SkyBlock
- Check if you're in the correct game mode
- Verify the mod is connected (`/cofl status`)
- Report persistent issues on Discord

### Performance Optimization

#### Reduce Lag
- Lower flip notification frequency
- Disable unnecessary features
- Reduce chat message length
- Use performance-friendly settings

#### Memory Usage
- Restart Minecraft periodically
- Close other applications
- Allocate more RAM to Minecraft
- Use OptiFine for better performance
