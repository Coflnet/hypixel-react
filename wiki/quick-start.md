# Quick Start Guide

Welcome to Coflnet! This guide will help you get started with our platform quickly.

## 1. Search for Items or Players

Use the search bar at the top of any page to find:
- **Items**: Type item names like "Aspect of the Dragon" or "AOTD"
- **Players**: Search by Minecraft username
- **Auctions**: Find specific auction UUIDs

### Search Tips
- Use partial names for faster results
- The search is ranked by popularity
- Recent searches are saved for quick access

## 2. Browse Auction History

Click on any item to see:
- **Price graphs** showing historical trends
- **Recent auctions** with detailed information
- **Active auctions** currently available
- **Price filters** to narrow down results

## 3. Set Up Notifications

Get alerts for:
- Price changes on specific items
- When players create auctions
- Auction outbids and sales

To create a notification:
1. Go to any item or player page
2. Click the "Notify" button
3. Set your criteria
4. Choose notification method

## 4. Use the Flipper

Find profitable auction flips:
1. Visit the [Flipper page](/flipper)
2. Log in with Google for full features
3. Customize your filters
4. Copy auction commands to use in-game

## 5. Install the Mod (Optional)

For the best experience, install our Minecraft mod:
1. Download from the [Mod page](/mod)
2. Install Forge 1.8.9
3. Place the mod in your mods folder
4. Use `/cofl help` in-game for commands
